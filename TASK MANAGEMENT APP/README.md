# TaskFlow - Task Management App

A fully responsive task management application with project checklists.

## Features

- **Project Management**: Create and organize tasks by projects
- **Task Checklist**: Add, edit, delete, and mark tasks as complete
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Dark/Light Mode**: Toggle between themes
- **Progress Tracking**: Visual progress indicators for projects
- **Priority Levels**: Categorize tasks as High, Medium, or Low priority
- **Due Dates**: Set deadlines for tasks with overdue indicators
- **Activity Log**: Track recent activities
- **Local Storage**: Data persists between browser sessions
- **Multiple Views**: List or grid view for tasks
